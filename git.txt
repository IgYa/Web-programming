пусто
===============================
Щоб видалити непотрібну гілку в Git:
git branch -d <назва_гілки>

Ця команда видалить гілку, але тільки якщо зміни з неї вже злиті з іншими гілками або віддаленим репозиторієм.

Якщо гілка ще не об'єднана і ви впевнені, що хочете видалити її разом із незбереженими змінами, використовуйте прапорець -D (великою літерою):

git branch -D <назва_гілки>

Зверніть увагу: Під час видалення гілки потрібно знаходитися в іншій гілці, відмінній від тієї, яку ви видаляєте.
